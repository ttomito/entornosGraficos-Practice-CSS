Ejercicio 1

1. ¿Qué es CSS y para qué se usa?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento HTML. Se usa para controlar el diseño, el formato y la apariencia visual de las páginas web, permitiendo a los desarrolladores separar el contenido (HTML) de la presentación (CSS).

2. CSS utiliza reglas para las declaraciones de estilo, ¿cómo funcionan?

Las reglas de CSS consisten en un selector y bloque de declaración. El selector identifica el elemento HTML al que se aplica el estilo, y la declaración especifica cómo debe verse ese elemento. Por ejemplo: h1 { color: blue; } aplica el color azul al texto de todos los encabezados de nivel 1.

3. ¿Cuáles son las tres formas de dar estilo a un documento?

Las tres formas de dar estilo a un documento son:

  1. Estilo en línea: Se aplica directamente al elemento HTML utilizando el atributo `style="..."`.
  2. Estilo interno: Se define dentro de la etiqueta `style` en la sección `head` del documento HTML.
  3. Estilo externo: Se define en un archivo separado con extensión ".css " y se vincula al documento HTML utilizando la etiqueta ``.

4. ¿Cuáles son los distintos tipos de selectores más utilizados?

Los selectores más utilizados son:

  • Selectores de elemento/etiqueta: Seleccionan elementos por su nombre, por ejemplo, `p` para todos los párrafos.
  • Selectores de clase: Seleccionan elementos por su atributo `class`, por ejemplo, `.destacado ` para elementos con la clase "destacado".
  • Selectores de ID: Seleccionan elementos por su atributo `id`, por ejemplo, `#principal ` para el elemento con el ID "principal".
  • Selector universal: `*` para seleccionar todos los elementos.

5. ¿Qué es una pseudo-clase? Cuáles son las más utilizadas aplicadas a vínculos?

Una pseudo-clase es un selector que permite aplicar estilos a elementos en función de su estado o posición en el documento. Las pseudo-clases más utilizadas aplicadas a vínculos/links son:

  • :link: Aplica estilos a los vínculos que aún no han sido visitados.
  • :visited: Aplica estilos a los vínculos que ya han sido visitados.
  • :hover: Aplica estilos cuando el usuario pasa el cursor sobre un vínculo.
  • :active: Aplica estilos cuando un vínculo está siendo activado (por ejemplo, al hacer clic).

6. ¿ Qué es la herencia?

La herencia en CSS se refiere a la capacidad de los elementos para heredar propiedades de estilo de sus elementos padres. Por ejemplo, si se define un color de texto en un elemento padre, los elementos hijos heredarán ese color a menos que se les asigne un valor diferente.

7. ¿ En qué consiste el proceso denominado cascada?

La cascada en CSS es el proceso mediante el cual el navegador determina qué estilos aplicar a un elemento cuando hay múltiples reglas que se aplican a ese elemento. El orden de importancia es: estilos en línea > estilos internos > estilos externos. CSS tiene en cuenta:

  • prioridad del selector
  • orden de aparición
  • herencia
  • especificidad
